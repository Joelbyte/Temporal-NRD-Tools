# Temporal-NRD-Tools
Some tools for preservation results of patterns, in the infinite-domain CSP setting. Two files: temporal_checker.py that checks whether a given temporal relation (represented as a list of orbits) is preserved by a given pattern, and classify_temporal.py that enumerates all ternary, temporal relations and checks if they are preserved by Maltsev, 3-NU, 3-EDGE, or 3-CUBE.
